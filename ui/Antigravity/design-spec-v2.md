# PixelNoise v2 设计规范文档

> 风格参考：Minecraft 物品栏 & 复古 8-bit 游戏 UI

---

## 一、设计系统 (Design System)

### 1.1 网格与对齐

| 规格     | 数值                                       |
| -------- | ------------------------------------------ |
| 基础网格 | `8×8px`（所有元素必须对齐）                |
| 方块单元 | `16×16px`（图标）/ `32×32px` / `64×64px`   |
| 间距系统 | `4px` / `8px` / `16px` / `32px` (严格倍数) |

### 1.2 色彩板

**背景层**

| 名称   | 色值      | 用途                   |
| ------ | --------- | ---------------------- |
| 主灰   | `#C6C6C6` | 日间主背景（石头灰）   |
| 护眼绿 | `#5C8C58` | 夜间模式背景（草地绿） |
| 深空蓝 | `#2C3E50` | 横屏沉浸底色           |

**方块/控件**

| 名称   | 色值      | 用途                    |
| ------ | --------- | ----------------------- |
| 原木色 | `#8B7355` | 未激活按钮、轨道        |
| 钻石蓝 | `#6B8E9F` | 激活/选中状态、滑块手柄 |
| 熔岩橙 | `#E8833A` | 播放按钮、强调状态      |
| 煤炭黑 | `#2D2D2D` | 文字、描边              |

**描边规范**

```
外描边: 2px solid #2D2D2D (所有方块必须有)
内高光: inset 0 1px 0 rgba(255,255,255,0.25) — 顶部高光，模拟凸起
内阴影: inset 0 -2px 0 rgba(0,0,0,0.25)     — 底部压暗，模拟厚度
按下态: 高光/阴影互换 + translateY(2px)
选中态: 内嵌反转（顶 2px 暗，底 1px 亮）
```

### 1.3 字体系统

| 角色            | 字体                   | 尺寸                 |
| --------------- | ---------------------- | -------------------- |
| Logo / 章节标题 | `Press Start 2P`       | `14px / 16px / 32px` |
| 正文 / 数值     | `VT323`                | `20px / 22px / 24px` |
| 像素图标        | Canvas 手绘 16×16 网格 | —                    |


## 三、组件库

### 3.1 基础方块按钮（48×48px）

```css
/* 默认 */
background: #8b7355;
border: 2px solid #2d2d2d;
box-shadow:
  inset 0 1px 0 rgba(255, 255, 255, 0.25),
  inset 0 -2px 0 rgba(0, 0, 0, 0.25);

/* 按下 */
transform: translateY(2px);
box-shadow:
  inset 0 2px 0 rgba(0, 0, 0, 0.25),
  inset 0 -1px 0 rgba(255, 255, 255, 0.25);

/* 选中 */
background: #6b8e9f; /* 钻石蓝 */

/* 强调（播放） */
background: #e8833a; /* 熔岩橙 */
```

### 3.2 像素滑块

| 元素         | 规格                                 |
| ------------ | ------------------------------------ |
| 轨道         | `100% × 8px`，原木色填充，2px 黑描边 |
| 填充区       | 钻石蓝，宽度跟随音量百分比           |
| 手柄（默认） | `16×24px`，钻石蓝，中心白色竖线纹理  |
| 手柄（激活） | 熔岩橙，宽度扩展至 `20px`            |
| 步长         | 5%（`Math.round(pct*20)*5`）         |

### 3.3 声音格子（Sound Cell）

```
┌────────────────┐
│   [16×16 图标]  │  ← Canvas 手绘像素图，居中
│   [12×12 ✓]    │  ← 右上角角标，选中态显示（钻石蓝底）
│ ───────────── │
│  底部标签条    │  ← 半透明黑底，VT323 16px 白色
└────────────────┘
 80×96px（竖屏）/ 72×76px（横屏抽屉）
```

### 3.4 像素图标绘制（16×16 网格）

| 音效 | 绘制规则                                          |
| ---- | ------------------------------------------------- |
| 雨声 | `#5B7FAA` 垂直短线，7 处错落排列，宽 2px 高 4px   |
| 风声 | `#7A9B76` 三条水平灰线，阶梯状排列                |
| 雪花 | `#B0C4DE` 十字(6×12)叠加(12×6)，四角各一 2×2 点   |
| 篝火 | 橙色三角底座(8×6) + 金黄顶焰(4×4) + 红色尖峰(2×4) |
| 海浪 | `#3C7A8A` 波浪路径 + `#5BA0B0` 第二层错位波浪     |
| 纯噪 | 随机散布 2×2 深浅灰色块（噪点）                   |

## 四、交互与动效

### 4.1 像素溶解过渡（Pixel Dissolve）

- **触发条件**：切换声音类型
- **实现**：`48×72` 逻辑分辨率 Canvas，方块按随机时序顺序涂满 → 执行状态变更 → 反向逐块清除
- **时长**：约 300-400ms（~30fps，批次渲染）

### 4.2 Canvas 粒子场景

| 音效 | 粒子特征                                     |
| ---- | -------------------------------------------- |
| 雨声 | 从顶部向右下斜落，`vx=1, vy=6`，2×4px 蓝色块 |
| 风声 | 从左侧横穿，`vx=3~5`，水平短条               |
| 雪花 | 缓慢曲折下落，`vy=1~2`，4×4px 浅蓝块         |
| 篝火 | 从中央底部向上漂移，正弦摆动，橙色 4×4px     |
| 海浪 | LFO 调制垂直位移，`vy=sin(x+t)`，蓝色横条    |
| 纯噪 | 每帧随机重置位置（雪花噪点效果）             |

- **帧率限制**：`t - lastFrame < 33ms`（≈30fps，保持复古感）
- **点击涟漪**：画布点击产生方块扩张涟漪，逐帧 `size += 2`，用 `strokeRect` 绘制

### 4.3 按钮反馈

- 按下时间：`transition: transform 60ms steps(1)` — 1 帧内生效
- 音频激活：浏览器保护策略限制，首次需用户手动点击播放

## 五、响应式规则

### 竖屏（`orientation: portrait` 或高度 > 500px）

- `flex-direction: column` 垂直堆叠
- 声音区：横向滚动条（`overflow-x: auto`）
- 控制栏：固定底部 `100px`

### 横屏（`orientation: landscape` 且 `max-height: 500px`）

- 标题栏：`position: absolute` 浮于顶部，透明背景
- 视觉区：`position: absolute; inset: 0` 占满全屏
- 声音条：隐藏（`display: none`），移入侧边抽屉
- 控制栏：`position: absolute; bottom: 0`，默认收起仅显 24px 细条

## 六、数据持久化

通过 `localStorage` key `pxn2` 保存以下字段：

```json
{
  "sound": "rain",
  "volume": 50,
  "theme": "day"
}
```

应用加载后 < 500ms 内自动恢复状态。

## 七、音频引擎（Web Audio API 纯合成）

| 音效 | 合成策略                              |
| ---- | ------------------------------------- |
| 雨声 | 粉噪 + 1000Hz 低通滤波                |
| 风声 | 粉噪 + 400Hz 带通滤波（Q=0.7）        |
| 雪花 | 粉噪 + 500Hz 低通 + Gain×0.6          |
| 篝火 | 褐噪 (Brownian) + 350Hz 低通          |
| 海浪 | 粉噪 + 低通 + 0.08Hz LFO 调制截止频率 |
| 纯噪 | 白噪直通                              |

> 零外部音频文件依赖，开箱即用。
