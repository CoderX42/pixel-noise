# PixelNoise 实现提示词（页面 / 工程）

将下列内容作为对 AI 或协作者的**实现规格说明**。**正式入口**为仓库根目录 **[`index.html`](../../index.html)**（单文件 HTML，内联 CSS + JS）。部署到 GitHub Pages 时根路径即可访问、**地址栏无 `.html`**。旧路径 `ui/PixelNoise.html` 仅作跳转页。

---

## 角色与目标

实现一个可在浏览器直接打开的**像素风环境白噪音**单页应用：**零外部音频文件**，全部用 **Web Audio API** 实时合成；画面用 **Canvas** 绘制像素粒子与场景装饰；UI 为**我的世界式方块 HUD**。需兼容桌面与手机（安全区、触摸、`AudioContext` 用户手势策略）。

---

## 技术约束

- **单文件交付**：根目录 `index.html` 内完成结构、样式、脚本，无构建步骤。
- **禁止**引用本地/远程 **音频资源**（wav/mp3 等）；允许 **Google Fonts**（如 Silkscreen、VT323）。
- **持久化**：`localStorage`（如键 `pxn5_v3`）保存 `sound`、`mixSounds`、`volume`、`theme`、`mixMode` 等。
- **画布**：按视口计算逻辑格网（如 `PX=4`，`CW/CH = ceil(inner/4)`），主循环约 **30fps**。
- **溶解过渡**：切换单场景时用独立 Canvas 层做像素块随机溶解。

---

## 音频引擎（按场景多层合成）

| 场景 | 合成要点 |
|------|----------|
| **雨声** | 粉噪 + 低通（雨幕）；随机高频 sine「水滴」；偶发远雷（棕噪/低频 burst，长间隔）；可与画面闪电联动。 |
| **森林** | 粉噪 + 低通（林风）；多路鸟鸣（频率扫描 / 啁啾 / 双音 / 颤音）；高频虫鸣（AM 颤音）。 |
| **海浪** | 粉噪 + LFO 调制低通；Gain LFO 潮汐感；周期性白噪 burst + 高通（浪花）；可选棕噪低频暗涌。 |
| **篝火** | 棕噪 + 低通底火；LFO 调制 gain（呼吸）；随机短白噪 burst + 高通（噼啪）；偶发中频爆裂。 |
| **风雪** | 粉噪 + bandpass（呼啸）+ 慢 LFO 阵风；高频层；偶发 sine 风啸 sweep。 |
| **纯噪** | 白噪 + 粉噪 + 棕噪按比例混合，听感饱满。 |

主音量经 **GainNode**，建议用平方曲线映射 0–100%。混音模式下对多路场景分别建链再汇到主增益。`stop` 时需断开/停止所有节点与 `setInterval`。

## Canvas 与交互

- **背景**：按主场景（或混音时取主层）绘制天空/地平线；**夜间**加像素星空；各场景装饰（雨：乌云、水洼、闪电；森林：树、草、萤火虫；海浪：多层浪、小岛、帆船；篝火：柴堆、石头、地面火光；雪：松树、积雪堆积；纯噪：CRT 扫描线与横线）。
- **粒子**：雨+飞溅、雪+堆积、火+火焰/火星/烟雾、浪+泡沫、森林+落叶与萤火虫、纯噪+颗粒。
- **点击画布**：按当前场景主色绘制方形涟漪（`strokeRect`），带透明度衰减。
- **播放中**：中央大播放按钮可在延时（如 1.5s）后以阶梯动画淡出隐藏；**暂停**时立即显示；底部播放键始终可用。

---

## 控件与逻辑

- **场景格**：横向 Hotbar（竖屏）与侧栏（横屏）；**MIX** 多选至多 5 路；溶解切换单场景。
- **播放**：中央 FAB + 底部按钮；图标用 **内联 SVG**（三角播放 / 双竖暂停），避免字体缺字。
- **移动端**：`pointerup` + `click` 去重；HUD **z-index** 高于画布；脉冲动画加在**内层 span**，避免 iOS `transform` 与点击区域错位。
- **音量**：经验条式滑块，5% 步进；**定时器**多档分钟，结束 `togglePlay(false)`。
- **快捷键**：空格播放/暂停；`1`–`6` 切场景；`↑` `↓` 音量。
