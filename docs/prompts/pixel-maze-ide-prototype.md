## BEGIN_PROMPT

你是一个资深前端工程师。请生成一个**完整的单页面应用（一个 HTML 文件即可，CSS/JS 内联）**，实现「**像素每日迷宫 · 七日挑战**」的**可运行原型**。

### 必须满足的功能

1. **每日种子**  
   - 使用用户**本地日期** `YYYY-MM-DD` 作为挑战 ID。  
   - 使用**确定性 PRNG**（如 mulberry32）：`seed = hash(日期字符串) ^ (level * 大质数)`，保证**同一天、同一关卡**迷宫完全一致。  

2. **七关递进**  
   - 关卡 1～7，每关一张**新迷宫**。  
   - 使用 **完美迷宫**（无环、任意两点唯一路径），算法任选：**递归回溯** / **Randomized Kruskal** / **Prim**。  
   - 难度递增：使用下面**固定奇数尺寸表**（宽×高，单位：格）：  
     - L1: 9×9, L2: 11×11, L3: 15×15, L4: 21×21, L5: 29×29, L6: 37×37, L7: 45×45  
   - 若 45×45 在弱设备上卡顿，可在代码里对 L7 做 `Math.min(45, maxSide)` 并注释说明。  
   - **起点**：第一行可走的通路格中靠左上一侧；**终点**：最后一行可走的通路格中靠右下一侧（与业务文档一致即可，实现时写死一种规则并在注释说明）。  

3. **操作与计步**  
   - **方向键 + WASD** 移动；**触摸**：提供屏幕下方四个大像素方向键（或滑动手势二选一，至少一种触摸方式）。  
   - **一步** = 成功移动到四连通相邻通路格；撞墙不计步。  
   - 每关显示**本关步数**；结算页显示**七关步数数组**与**总步数**。  

4. **通关与流程**  
   - 到达终点 → 简短提示 → 进入下一关；第 7 关结束 → **结算页**。  
   - 结算页：输入**昵称**（可选，默认「匿名」）、**提交**写入排行榜。  
   - **重玩今日**：确认后清空进度，迷宫仍为今日种子不变。  

5. **本地排行榜**  
   - `localStorage`，键名：`maze7_rank_<YYYY-MM-DD>`。  
   - 每条：`{ id, nickname, totalSteps, stepsByLevel: number[7], submittedAt }`。  
   - 仅**完成七关**后可提交；按 **totalSteps 升序**，同分按 **submittedAt 升序**；最多保留 **20** 条。  

6. **进度恢复**  
   - 使用 `localStorage` 保存当日进行中 Run（当前关、当前关步数、已过关步数列表）；刷新页面可继续。  
   - **换日**：若检测到日期与保存不一致，清空旧进度并开始新日（简单处理即可）。  

7. **像素风 UI**  
   - **Canvas 2D** 绘制迷宫；`image-rendering: pixelated`；每格用矩形绘制墙/路。  
   - 字体：从 Google Fonts 引入 **Silkscreen** 或 **Press Start 2P**（标题）+ **VT323**（数字）。  
   - 按钮：**方块描边 + 按压位移**，类似 Minecraft HUD。  
   - 显示：`第 k/7 关 · 今日 YYYY-MM-DD`。  

8. **算法校验**  
   - 生成后用 **BFS** 验证起点到终点可达；计算 **optimalSteps**（最短步数）。  
   - 在结算页或关底**小字显示**「本关最短：X 步」供对比（可选但建议做）。  

### 技术约束

- **不依赖**构建工具；单 HTML，双击或静态服务器均可打开。  
- **无后端**；无外部 API。  
- 代码结构清晰：分区注释 `// --- SEED ---` `// --- MAZE ---` `// --- RENDER ---` `// --- UI ---`。  
- 控制台无报错；现代移动浏览器可玩。

### 交付物

- 输出**完整可保存的 `index.html` 文件内容**（或若 IDE 支持多文件，仅一文件即可）。  
- 在文件顶部用 HTML 注释写 5 行以内的「如何本地运行」说明。

### 不要做

- 不要引入 React/Vue 等大型框架（原型保持单文件）。  
- 不要做全局联网排行榜。  
- 不要生成占位符 TODO 而不实现核心循环。
